# s_toggle_example

This example demonstrates basic and advanced usage of the `s_toggle` package.

## Run

```bash
flutter pub get
flutter run
```

## What it shows

- Basic toggle with custom colors (green ON, red OFF)
- Reading and reacting to the toggle state via `onChange`
- Custom sizing and animation duration
